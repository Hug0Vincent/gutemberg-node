Context

Ille-et-Vilaine archives' mission is to keep track of all the departmental records and make them available to the public. Since they have to work with a huge amount of data and a lot of documents types, Ille-et-Vilaine archives have started a digitalization campain. Project Gutemberg was initiated so people can easily access all these documents on a common platform.

  • Amount Big amount of data
  • Plurality Lots of document types
  • Consultation Available to the public
  • Ergonomics Easy search and consultation
  • Collaborative Share knowledge

The application

Project Gutemberg is a web application that aims to make consulting Ille-et-Vilaine' digitalized documents available to the public. The application is content based thanks to its clean and responsive user interface which makes it easy to consult documents even on a smartphone or a tablet.

Gutemberg is a multi-documents application, it allows to make available different types of document : old press, registration register, register of civil status, ...

Tired of spending hours to find the document you need ? Gutemberg's search engine allows you to easily and efficiently find all the documents you need. The application also has an advanced search tool which allows you to refine your query.

Thanks to its responsive and clean interface, Gutemberg is an elegant way of consulting all types of document. Consulting is easy and intuitive, using a mouse or a touch device.

Gutemberg allows its users to share their knowledge thanks to its collaborative annotations system. With this system, you'll be able to share and compare your interpretations with all the users.

The application allows every user to register so he can contribute to the projet by annotating documents or by correcting current annotations. Those will then be visible by every Gutemberg's user.

This user system allows the Ille-et-Vilaine archives to offer a really collaborative platform to their readers while still having moderators and administrators so things don't get messy.

Playlists allow users to make a ordonned list of annotations of different documents. Reader can then use these playlists to go through documents of a predefined theme.
